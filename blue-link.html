🔵 Blue Links — Ateneo de Davao Alumni Portal

A responsive desktop-first web app for AdDU alumni. Features job board with live filtering, application tracking, verified alumni credentials, and stealth mode privacy.

📋 Prerequisites

This project is a single HTML file with zero build dependencies. You only need one of the following to run it:

  • Any modern web browser (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
  • Optional: A local static file server for the best experience

🚀 Quick Start (Simplest Method)

Option 1 — Double-click (Instant, no setup):

1. Download or clone this repo
2. Navigate to the project folder
3. Double-click blue-links.html
4. Opens directly in your browser ✓

Option 2 — VS Code Live Server (Recommended for development):

1. Install Visual Studio Code
   → https://code.visualstudio.com

2. Install the "Live Server" extension
   → Open VS Code → Extensions (Ctrl+Shift+X)
   → Search "Live Server" by Ritwick Dey → Install

3. Open the project folder in VS Code
   → File → Open Folder → select blue-links folder

4. Right-click blue-links.html in the file tree
   → Click "Open with Live Server"
   → Browser opens at http://127.0.0.1:5500 ✓

Option 3 — Python (Built into macOS/Linux):

# Python 3 (macOS / Linux / Windows with Python installed)
cd path/to/blue-links
python3 -m http.server 8080
# Open: http://localhost:8080/blue-links.html

# Python 2 (older systems)
python -m SimpleHTTPServer 8080

Option 4 — Node.js serve (if Node is installed):

# Install serve globally (one-time)
npm install -g serve

# Run from project folder
cd path/to/blue-links
serve .
# Open the URL shown in terminal ✓

🌐 Deploying to GitHub Pages (Free Hosting)

1. Create a GitHub account at github.com
2. Create a new repository named: blue-links

3. Upload files:
   git init
   git add .
   git commit -m "Initial commit — Blue Links portal"
   git remote add origin https://github.com/YOUR_USERNAME/blue-links.git
   git push -u origin main

4. Enable GitHub Pages:
   → Repository Settings → Pages
   → Source: Deploy from a branch
   → Branch: main / root
   → Save

5. Your app is live at:
   https://YOUR_USERNAME.github.io/blue-links/blue-links.html ✓

🔐 Verification Hash Technology

The app uses the Web Crypto API (built into all modern browsers — no library needed) to generate a real SHA-256 hash of alumni credentials on login. In production, this hash would be issued by the University Registrar and stored in a signed JWT or on-chain ledger.

// useVerificationHash — composable logic
const data = encoder.encode(`${alumniId}|${year}|${degree}`)
const hash = await crypto.subtle.digest('SHA-256', data)
// Formats to: ALM-2026-SJ-XXXX-XXXXX

📱 Responsive Breakpoints

  • 1100px — Context panel narrows
  • 900px — Right context panel hides; single column layouts
  • 768px — Sidebar hides; mobile job list / detail switch
  • 480px — Compact mobile: topbar search hides, 2-col grids

This project was architected with Claude (Anthropic) as Lead Vue.js Engineer consultant. Key decisions — three-column desktop layout, Web Crypto verification, Stealth Mode state management, and inline filter chips — were proposed by the AI based on UX analysis of the mobile design system.
